NextLive Shop Architecture

คู่มือสรุปสถาปัตยกรรมระบบ Serverless แบบ Full-Stack การติดตั้ง โครงสร้างรายได้ และข้อจำกัดเชิงเทคนิคของแพลตฟอร์มอีคอมเมิร์ซและเอนเตอร์เทนเมนต์

ภาพรวมระบบนิเวศ (System Ecosystem)

NextLive ขับเคลื่อนด้วยสถาปัตยกรรมไร้เซิร์ฟเวอร์ (Serverless) โดยผสานรวมบริการชั้นนำเพื่อลดต้นทุนการดูแลรักษาและเพิ่มความเสถียร

🌐

GitHub Pages (Frontend)

หน้าบ้าน (UI) ของผู้ใช้และแอดมิน โฮสต์ฟรีด้วย HTML, TailwindCSS และ Vanilla JS รองรับการแสดงผลแบบ SPA

⚙️

Google Apps Script (Backend)

ทำหน้าที่เป็น API หลัก รับมอบหมายในการประมวลผลโลจิก การตัดสต๊อก และส่งอีเมลแจ้งเตือน (จำกัดโควต้า 100-500 ฉบับ/วัน)

🔥

Firebase Realtime DB

ฐานข้อมูลหลักความเร็วสูง เก็บประวัติผู้ใช้ ออเดอร์ รวมถึงระบบ Live Chat ที่ซิงค์ข้อมูลกับหน้าจอผู้ใช้และแอดมินแบบ Realtime

📁

Google Drive (Storage)

ระบบจัดเก็บไฟล์แนบ สลิปโอนเงิน รูปภาพจากแชท โดยใช้พื้นที่โควต้าฟรี 15GB ของบัญชี Google

สัดส่วนองค์ประกอบโครงสร้างระบบ

🛣️ เส้นทางการติดตั้งระบบ (Deployment Journey)

คลิกเลือกแต่ละขั้นตอนเพื่อดูคำแนะนำเชิงเทคนิคในการติดตั้งระบบ NextLive ให้สมบูรณ์แบบตั้งแต่ต้นจนจบ

โมเดลรายได้และคอมมิชชั่น (Affiliate Hierarchy)

ระบบกระจายรายได้แบบ 4 ชั้นลึก (Multi-tier) ที่ช่วยกระตุ้นการบอกต่อ กราฟด้านล่างแสดงอัตราส่วนแบ่งรายได้แบบลดหลั่นตามระดับชั้นการแนะนำ

Tier 1 (ตรง) : 5%

ค่าคอมมิชชั่นเมื่อผู้ใช้ที่เราแนะนำโดยตรงทำการสั่งซื้อหรือฝากเงิน

Tier 2 (ลูกโซ่) : 1%

ค่าคอมมิชชั่นเมื่อเครือข่ายชั้นแรกไปแนะนำผู้ใช้อื่นต่อ

Tier 3 (ระดับลึก) : 0.5%

ส่วนแบ่งรายได้ต่อเนื่องจากเครือข่ายชั้นที่สอง

Tier 4+ (เครือข่ายฐาน) : 0.1%

รายได้จากการขยายฐานแบบไม่จำกัดความลึกของเครือข่าย

* กรณีผู้ซื้อเป็น "Guest" (ไม่ได้สมัครสมาชิก) ผู้แนะนำจะได้รับค่าคอมมิชชั่นแบบเหมาที่อัตรา 1% ทันที

⚠️ ขีดความสามารถและข้อควรระวัง (Constraints)

สรุปจุดแข็งและข้อควรระวังที่แอดมินต้องรับทราบในการบำรุงรักษาระบบ Serverless ตามที่ระบุไว้ในคู่มือ

การประเมินประสิทธิภาพระบบ

🔓

ข้อควรระวัง: ฐานข้อมูล Firebase

มีการตั้งค่า Rules .read: true, .write: true เพื่อรองรับ Guest Chat. ห้ามเก็บข้อมูลบัตรเครดิตหรือข้อมูลอ่อนไหวทางกฎหมายในฐานข้อมูลนี้เด็ดขาด

💾

ข้อจำกัด: พื้นที่จัดเก็บข้อมูล

Google Drive ให้พื้นที่ฟรี 15GB ไฟล์สลิปและไฟล์แชทจะกินพื้นที่ แอดมินต้องทำการ เคลียร์สลิปเก่าๆ ทุกๆ 1-3 เดือน เพื่อป้องกันระบบล่ม

⏳

ข้อจำกัด: โควต้าอีเมลรายวัน

ฟังก์ชัน Broadcast และการส่งใบเสร็จผ่าน GAS บัญชีฟรีจำกัดที่ 100-500 ฉบับต่อวัน หากเกินนี้ควรพิจารณาใช้ Google Workspace
